# Student Record Manager (Java)

A simple console-based Java application to manage student records.  
This project demonstrates core Java concepts and clean coding practices suitable for a fresher Associate Software Engineer role.

## Features
- Add student details
- View all students
- Update student information
- Delete student records
- Menu-driven console interface

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the main class
